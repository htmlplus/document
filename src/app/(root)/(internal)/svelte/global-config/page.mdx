# Global Config In Svelte

Sets a config for the `size` property of the `spinner` element.

```js
import App from './App.svelte';

// Setup the global config.
import { setConfig } from '@htmlplus/ui/config.js';
setConfig({
  elements: {
    'plus-spinner': {
      properties: {
        size: {
          default: 'lg'
        }
      }
    }
  }
});

const app = new App({
  target: document.body
});

export default app;
```
